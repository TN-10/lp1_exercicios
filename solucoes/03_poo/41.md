### Problema

41 - Numa classe ou método genérico como podemos inicializar o tipo genérico
com o seu valor por omissão (equivalente a zero ou `null`)?

## Solução

Usamos a keyword `default`.

``` cs

public class AindaOutraClasse<T1, T2> {
	private T1 val1; private T2 val2;
	public AindaOutraClasse(T1 val1) {
		this.val1 = val1;
		this.val2 = default(T2);
	}
}
```
